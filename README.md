# WIK-DPS-TP01*

```
Commande pour lancer le projet:
- npx tsc
- node index.js

Ma variable d'envirronement est 3000.
Ce code est conçu pour créer un serveur HTTP simple qui écoute les requêtes GET sur l'URL /ping. Lorsqu'une telle requête est reçue, le serveur renvoie les en-têtes de la requête au format JSON. Si la requête ne correspond pas à ces critères, le serveur renvoie une réponse avec le code d'état 404 Not Found.

```
